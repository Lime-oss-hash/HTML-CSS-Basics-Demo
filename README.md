# HTML & CSS Basics Demo

A minimal, self‑contained site showcasing core HTML elements and beginner‑friendly CSS patterns.

## Overview
- Uses semantic HTML (`header`, `nav`, `main`, `section`, `footer`).
- Demonstrates text tags, `div` vs `span`, forms, images, links.
- Shows essential CSS: selectors, flexbox layout, cards, sticky nav, and media queries.

## Quick Start
- Open `index.html` in any modern browser.
- Optional local server:
  - Python: `python -m http.server 8000`
  - Node: `npx serve`
  - Then visit `http://localhost:8000/`.

## What’s Included
- Text tags and inline vs block elements.
- Semantic layout with a top navigation.
- Flexbox row with responsive wrapping.
- Accessible form with labels, inputs, select, focus styles.
- Image with responsive sizing and an external link style.
- Simple reset, container, and card components.
- Mobile‑friendly spacing via a media query at 700px.

## Explore the Demo
- Use the top nav to jump to: `#text`, `#layout`, `#form`, `#media`.
- Resize the browser to see the media query effects.
- Click “Send” on the form to trigger the demo alert.

## Extend It
- Add more boxes to the flex container and observe wrapping.
- Tweak the breakpoint in `styles.css` to learn media queries.
- Create a new section with elements you want to practice.
